# Flutter Business Card 📱

Aplicativo desenvolvido com **Flutter** que apresenta um **cartão de visita digital**, exibindo informações profissionais como nome, profissão e formas de contato.

Este projeto foi criado com o objetivo de praticar conceitos básicos de desenvolvimento mobile com Flutter e construção de interfaces simples.

---

## 🚀 Tecnologias utilizadas

* Flutter
* Dart
* Material Design

---

## 📱 Funcionalidades

* Exibição de foto de perfil
* Nome e cargo profissional
* Informações de contato (telefone, e-mail)
* Layout simples e responsivo

## ▶ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/maisamarall/flutter-business-card.git
```

2. Acesse a pasta do projeto

```
cd flutter-business-card
```

3. Instale as dependências

```
flutter pub get
```

4. Execute o aplicativo

```
flutter run
```
